This application is intended for use by PostalAnnex.

Developed solely by Daniel C. Senn unless otherwise specified.

Functionality Requirements:

- Ability to scan packages and log them in a spreadsheet.
- Scanned barcodes will automatically detect their carrier type based on regular expressions (only works for USPS, UPS, FedEx, DHL)
- Package log will have a search function to find packages by date or by box number.
- Box number will be entered manually for each package.
- Signature fields for pick up and for initialling the delivery of packages by attendants.
